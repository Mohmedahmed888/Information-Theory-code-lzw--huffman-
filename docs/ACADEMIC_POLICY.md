# Academic integrity & licensing (read before submissions)

This project ships with an **MIT** license in the repository root (`LICENSE`). That license governs **code reuse by others**.

**Important for coursework:** your university may require additional statements about authorship, collaboration, plagiarism policies, or a specific license embargo. Requirements vary by instructor and jurisdiction.

Recommended checks:

1. Confirm with your syllabus / TA whether submitting to a **public GitHub repository** is allowed.  
2. If anonymity is required, remove or anonymize identifiable metadata (names inside `LICENSE`, `pyproject.toml`, README links, etc.).  
3. Follow your institution’s rules on AI-assisted tooling disclosure (if applicable).

Nothing in this file overrides your faculty’s formal requirements.
